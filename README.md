# TaskHawk Website Starter

This is a static starter ready for Vercel deployment and Apple Universal Links.

## Files

- `index.html`: main landing page
- `styles.css`: visual styling
- `app.js`: basic client interactions
- `.well-known/apple-app-site-association`: Universal Links mapping
- `vercel.json`: header config so AASA serves as JSON

## Next steps

1. Replace placeholder copy with your real content.
2. Update `TEAMID.com.yourcompany.yourapp` in `.well-known/apple-app-site-association`.
3. Deploy repo to Vercel.
4. Connect your GoDaddy domain in Vercel.
5. Add Associated Domains in Xcode:
   - `applinks:yourdomain.com`
   - `applinks:www.yourdomain.com`
